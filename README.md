## Usage

```sh
git clone https://github.com/seungyup26/sydosnet && cd sydosnet
```

```sh
g++ linux/src/main.cpp linux/src/lib/dosapi.cpp -o test/sydosnet.exe -pthread && cd test
```

```sh
./sydosnet.exe -h
```
